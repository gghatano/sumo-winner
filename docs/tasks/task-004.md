# Task 004: テキスト生成（投稿用フォーマット整形）

## 概要
予想入力に基づいて、投稿用テキストをリアルタイム生成する。

## ブランチ / worktree
- `gitworktree/feature-004-text-format`

## 依存タスク
- task-001（プロジェクト雛形）

## ブロックしているタスク
- task-005（クリップボードコピー）

## 作業内容

1. **テキスト整形ユーティリティ** (`src/lib/format.ts`)
   - `generatePredictionText(day, matches, predictions): string`
   - ヘッダ行: 日目に応じた文言（例: `十二日目（・ω・）ノ`）
     - 日目 → 漢数字変換ユーティリティ
   - 各行: 予想に応じて `○` `●` を付与
     - 東勝ち予想: `○東力士名－西力士名●`
     - 西勝ち予想: `●東力士名－西力士名○`
     - 未選択: ` 東力士名－西力士名 ` （記号なし）

2. **プレビューコンポーネント** (`src/components/PredictionPreview.tsx`)
   - テキストエリアに生成結果を表示
   - 予想変更に応じてリアルタイム更新

3. **テスト**
   - format.ts のユニットテスト
   - 日目漢数字変換のテスト
   - 各パターン（東勝ち/西勝ち/未選択）のテスト

## 完了条件
- 予想入力に応じて正しいフォーマットのテキストが生成される
- ヘッダ行に正しい日目（漢数字）が表示される
- 未選択の取組も行として含まれる（記号なし）
