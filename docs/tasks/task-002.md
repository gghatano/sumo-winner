# Task 002: JSON読み込み + 場所/日目セレクタ + 取組一覧表示

## 概要
JSONデータを読み込み、場所・日目を選択するUIと取組一覧を表示する。

## ブランチ / worktree
- `gitworktree/feature-002-match-display`

## 依存タスク
- task-001（プロジェクト雛形）

## ブロックしているタスク
- task-005（コピー機能は全UI揃ってから統合）

## 作業内容

1. **データ取得ユーティリティ** (`src/lib/api.ts`)
   - `latest.json` を取得して最新場所・日目を特定
   - `{YYYYMM}/{day}.json` を取得して取組一覧を返す
   - 型定義: `Basho`, `Match`, `TorikumiData`

2. **場所/日目セレクタコンポーネント** (`src/components/BashoSelector.tsx`)
   - 場所セレクト（例: 2026年1月場所）
   - 日目セレクト（初日〜千秋楽 = 1〜15）
   - デフォルトは最新場所・最新日

3. **取組一覧コンポーネント** (`src/components/MatchList.tsx`)
   - 各取組を東力士名/西力士名で表示
   - 予想ボタン（東○ / 西○ / 未選択に戻す）のUI配置
   - ボタンクリック時のコールバック（状態管理はprops経由）

4. **App.tsx統合**
   - セレクタ変更 → JSON再取得 → 一覧更新の流れ
   - ヘッダにデータ更新日時を表示

## 完了条件
- 場所・日目を変更すると対応するJSONが読み込まれ取組が表示される
- デフォルトで最新場所・最新日が選択されている
- 各取組に東/西の予想ボタンが表示される
- 予想ボタンをクリックすると選択状態が変わる（この時点ではメモリ上のみでOK）
